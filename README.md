🧬 Proyek Unggulan (Update 30 Hari Terakhir)

🔹 AstroBio Insight Navigator — in progress

Dasbor AI interaktif untuk meringkas 608+ publikasi biosains NASA dan sumber terkait. Dirancang bagi ilmuwan, manajer program, dan arsitek misi untuk:

🔬 Memetakan dampak eksperimen biologis ruang angkasa pada kesehatan manusia, tumbuhan, dan sistem pendukung kehidupan.
🧩 Mengungkap celah pengetahuan dan konsensus ilmiah.
🌍 Menautkan publikasi ke data primer, metadata eksperimen, pendanaan, dan hasil misi.
Sorotan fitur:

🤖 AI Summarization Hub: ringkasan multi-level (headline, paragraf, poin) per publikasi; ekstraksi berbasis bagian (Pendahuluan–Metode–Hasil–Kesimpulan); highlight temuan kritis untuk misi Bulan/Mars.
🧠 Knowledge Graph Explorer: jaringan interaktif antar publikasi, topik, eksperimen OSDR, dan proyek hibah; filter Organisme (Manusia/Tumbuhan/Mikroba), Lingkungan (Mikrogravitasi/Radiasi), Sistem (Kardiovaskular/Neurologis/dll).
🚀 Mission Scenario Simulator: pilih skenario (mis. “Mars Surface Habitat 2035”) → rekomendasi riset, risiko, dan gap prioritas.
👩‍🚀 Persona-driven UI: Mode Ilmuwan, Manajer, Arsitek Misi dengan KPI, visual, dan rekomendasi adaptif.
🔍 Insight Feed & Citation Trail: linimasa temuan penting + tautan ke teks lengkap, dataset OSDR, dan NASA Task Book.
Stack (target): TypeScript · Next.js · FastAPI · Python · LLM · Vector DB · Graph DB · D3.js

Status: MVP aktif dikembangkan — kontribusi dan kolaborasi dipersilakan.
